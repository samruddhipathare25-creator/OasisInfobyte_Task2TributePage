 Project Title: Tribute Page Website
 Project Description:

The Tribute Page is a simple and responsive web page created using HTML ,Script and CSS to honor and showcase the life, achievements, and contributions of a great personality. This project is commonly developed as a beginner-friendly web development task to practice webpage structure, styling, and content presentation.

The page typically includes a title, image, biography section, timeline of achievements, and a concluding message that highlights the importance of the person’s contributions.

 Objectives:

To design a clean and structured webpage.
To practice HTML elements like headings, paragraphs, images, and lists.
To improve CSS styling and layout skills.
To create a responsive design for different screen sizes.

 Features:

 Profile image of the personality
 Short biography section
 Timeline of major achievements
 Attractive and responsive design
 External link for more information

Technologies Used:

HTML5 – Page structure
CSS3 – Styling and layout
Script - Animation
